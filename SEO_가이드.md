## 1. 구현된 SEO 요소
- **총 25개 기사** (전 기사 1,500~2,500자 본문 + 글마다 監修 코멘트 박스 포함)
- **지역(集患) 키워드 기사 5개**: `インプラント 新宿/渋谷/銀座/池袋/横浜` → '에리어에서 찾기' 카테고리
- **정적 HTML 페이지** (기사당 1파일) — SPA 대신 SSR식 정적 페이지로 크롤링·색인 최적화
- 페이지별 고유 `<title>` / `meta description` / `meta keywords`
- **구조화 데이터(JSON-LD)**: `MedicalWebPage` + `BreadcrumbList` + `FAQPage`(해당 글) + 홈 `WebSite`/`ItemList`
- OGP / Twitter Card / canonical 태그
- H1→H2→H3 시맨틱 구조, 목차(앵커링크), 카테고리 내부링크·관련글 링크
- `sitemap.xml` / `robots.txt`
- 監修者(감수자)·공개/갱신일·면책문구 → 의료 YMYL의 **E-E-A-T** 신호
- 이미지는 **인라인 SVG**(경량·확대무손실·저작권 안전, `aria-label` 포함)

## 3. ⚠️ 의료광고가이드라인(医療広告ガイドライン) 주의
일본 의료법상 **비교우량광고(比較優良広告)**는 사실이라도 금지됩니다.
- 금지 예: `日本一` `地域No.1` `最高の治療` `ベスト10 병원` `A병원보다 저렴`
- 그래서 요청하신 '임플란트 베스트10'은 **실명 순위 대신 '선택법 10가지 체크포인트'** 형식으로 제작했습니다 (검색량은 유지, 규제는 회피).
- 실명 병원 랭킹이 꼭 필요하면, (a) 광고가 아닌 객관적 정보로 한정, (b) 한정해제(限定解除) 요건 충족, (c) 법무 검토를 권장합니다.
- 체험담·비포애프터 사진도 규제 대상 → 본 블로그는 일부러 배제했습니다.
